Cheat sheet Git


Commande de base git


git init = permet d'initialiser un dossier crée
git config –global user.email = permet d'enregistrer un email
git config –global user.name = permet d'enregistrer son nom
git config –global color.ui true
git config –list = permet de voir tout ce qui a été défini
git status = permet de voir les dossiers référencés
git add = ajoute les fichier
git add *.html= permet d'ajouté tout les fichier avec l'extention .html
git commit -m « message » = permet d'jouter un commit de ce que l'on viens de faire
git log = permet de voir tout les commits effectués
git log -n x = permet de voir les x derniers commit, exemple git log -n 3 permettra de voir les 3 dernier commits
git log –oneline = permet d'afficher les infos sur une seul ligne
git log « nom du fichier » = permet de voir toute les modif sur le fichier demandé
git log -n x -p « nom du fichier » = affiche les information complète du fichier, -n x permet aussi de filtrer les x dernières lignes
git checkout +clé = permet de revenir a l’état donné
git diff = permet de voir les modification sur le fichier
git push -u origin master = permet d'envoyer les fichier de ungit sur gitub
